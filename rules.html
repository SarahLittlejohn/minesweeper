Minesweeper

The rules or the game:

  • Select Play to start a game.
  • You want to avoid the mine.
  • Select a box to guess whether it is clear from a mine.
  • If the box is a mine, it will turn red and the game will be over.
  • If the box is directly next to the mine, it will turn dark orange.
  • If the box is two boxes away from a mine, it will turn light orange.
